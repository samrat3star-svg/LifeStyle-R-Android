## ðŸ”„ Sync Workflow for OneDrive

1. **Pause OneDrive Sync** while actively coding/building to avoid file lock errors with `build/` directories.
2. **Resume Sync** when finished to upload changes to the cloud.
3. On the **Laptop**, open the project and let Android Studio perform a **Gradle Sync**.

    - If "File not found" errors occur, do `Build > Clean Project` then `Rebuild Project`.
